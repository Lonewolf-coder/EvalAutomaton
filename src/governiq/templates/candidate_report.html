How This Evaluation Was Conducted

Pipeline A: Structural Analysis

Your bot export JSON was parsed to extract all dialogs, entity nodes, service nodes, Agent Nodes, and FAQ entries. Each was compared against the assessment requirements defined in the manifest. This analysis is deterministic — running it again on the same export produces identical results.

Pipeline B: Live Conversation Testing

An AI-powered conversation driver connected to your bot's webhook and conducted real conversations — booking appointments, retrieving records, modifying data, and testing edge cases. After each conversation, the system verified what data actually reached your Mock API.

Scoring: CBM Structural (40%) + Webhook Functional (40%) + Compliance (10%) + FAQs (10%). Pass threshold is 70% with no critical compliance failures. Both pipelines are independent — a structural pass with a webhook fail (or vice versa) still counts toward the total score.
